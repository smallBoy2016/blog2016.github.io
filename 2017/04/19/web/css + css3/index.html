CSS常用选择器

选择符：

1.元素选择符

  1. 通配选择符
    *所有元素对象
  2. 类型选择符
    以文档语言对象类型作为选择符
  3. id选择符
    以唯一标识符id属性作为选择符
  4. class选择符
    以class名作为选择符

2.关系选择符

  1. 包含选择符

    E F

  2. 子选择符

    E>F

  3. 相邻选择符

    E+F 选择紧贴在E元素之后F元素。

  4. 兄弟选择符

    E~F选择E元素所有兄弟元素F。

3.属性选择符

4.伪类选择符

  • 匹配父元素的第一个子元素E

    E:first-child{height:10px;}

  • 匹配父元素的最后一个子元素E

    E:last-child

  • 匹配父元素的第n个子元素E

    E:nth-child(n){xxx}

常用class的命名：

(1)颜色:使用颜色的名称或者16进制代码,如

.red { color: red; }
.f60 { color: #f60; }
.ff8600 { color: #ff8600; }

(2)字体大小,直接使用”font+字体大小”作为名称,如

.font12px { font-size: 12px; }
.font9pt {font-size: 9pt; }

(3)对齐样式,使用对齐目标的英文名称,如

.left { float:left; }
.bottom { float:bottom; }

(4)标题栏样式,使用”类别+功能”的方式命名,如

.barnews { }
.barproduct { }
